# Car Detailent - Sitio Informativo

Este repositorio contiene los archivos de una página web informativa para usuarios destinada a presentar los servicios de detalle automotriz de "Car Detailent".

Contenido y propósito:
- `index.html`: estructura HTML completa con secciones (hero, beneficios, paquetes, galería) preparada para estilos y recursos.
- `assets/css/styles.css`: estilos base y reglas responsive (grid y flexbox).
- `assets/js/main.js`: pequeñas interacciones (año dinámico y smooth scroll).
- `assets/img/`: lugar para colocar las imágenes (hero, muestras antes/después, iconos si aplica).

Cómo usar / previsualizar:
1. Coloca las imágenes en `assets/img/` con los nombres usados en `index.html` (p. ej. `car-blue.jpg`, `g1-before.jpg`, `g1-after.jpg`, etc.).
2. Abre `index.html` en un navegador local (doble clic o servidor estático) para revisar el resultado.

Notas:
- El proyecto está pensado como una página informativa para usuarios; puedes ajustar tipografías, imágenes y texto según la marca.
- Los iconos sociales y de tarjetas ya están embebidos como SVG inline para facilitar estilizado.

Si quieres, puedo añadir un script `serve` simple o instrucciones para un servidor local (`npx http-server` o `python -m http.server`).
